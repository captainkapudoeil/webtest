Insertion d'image!! :D

Il faut savoir adapter le format a l'image que l'on met!!
Plusieurs formats donc :

  • JPEG

    Conçu pour réduire le poid de photos de bonnes qualitées, ses extensions sont ".jpg" ou ".jpeg"

  • PNG

    Conçu pour le réduire le poid des graphiques, il est conccurent des GIF.
    2 versions :

    • 8 bits = 256 couleurs
    • 24 bits = 16 millions de couleurs !!!!
    Il a également la capacité de gérer la transparence des images

  • GIF

    256 couleurs seulements mais permet de créer des images animées
